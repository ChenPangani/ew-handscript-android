### 2.1 启动页（Splash Screen）

#### 2.1.1 布局结构

```
全屏尺寸：360dp × 760dp
背景：纯色 #1A1A2E（玄青）渐变至 #2D3561（靛蓝）

层级结构（从底到顶）：
Z-0:  背景渐变层（全屏）
Z-1:  灵气粒子层（Canvas全屏，见动效定义）
Z-2:  中央Logo容器（120dp × 120dp，居中）
Z-3:  品牌文字"蚯蚓·手书修仙传"（Display字体，32sp）
Z-4:  副标题"笔迹成仙，字炼永生"（Caption字体，14sp，#8A8680）
Z-5:  版本号 "v0.1"（Caption，12sp，#8A8680，底部距底32dp）
```

#### 2.1.2 元素位置与尺寸

| 元素 | 位置 | 尺寸 | 其他 |
|------|------|------|------|
| Logo图标 | 居中偏上（Y = 280dp） | 120dp × 120dp | 蚯蚓盘卷成太极图造型，赤金描边 |
| 品牌名 | Logo下方16dp | wrap_content | 字色 `#FFD700` 赤金 |
| 副标题 | 品牌名下方8dp | wrap_content | 字色 `#8A8680` 中灰 |
| 版本号 | 底部32dp | wrap_content | 居中对齐 |

#### 2.1.3 动效定义

```yaml
动画名称: 灵气汇聚显形

时长策略:
  首次安装启动: 3000ms（完整动画，不可跳过）
  后续冷启动:   1500ms（精简动画，允许点击跳过）
  热启动:       0ms（直接显示首页）

跳过机制（仅后续冷启动生效）:
  - 触发: 用户点击屏幕任意位置
  - 行为: 立即执行阶段4退出动画（淡入首页）
  - 最低保证: 至少展示 Logo 500ms 后方可跳过

首次安装启动 — 完整版（3000ms）:

  阶段1（0-800ms）:
    - 背景: 纯色 #1A1A2E 无渐变
    - 粒子: 屏幕四周向中心汇聚，速度 2dp/frame
    - Logo: opacity=0, scale=0.5

  阶段2（800-1800ms）:
    - 背景: 渐变从中心向四周扩散，#1A1A2E → #2D3561
    - 粒子: 加速汇聚（4dp/frame），在Logo位置形成光晕
    - Logo: opacity 0→1 (ease-out, 400ms), scale 0.5→1.1 (spring, 600ms)

  阶段3（1800-2600ms）:
    - Logo: scale 1.1→1.0 (spring, bounce=0.2, 400ms)
    - 品牌名: opacity 0→1 (ease-out, 300ms), Y偏移 20dp→0
    - 副标题: opacity 0→1 (ease-out, 300ms, delay=200ms), Y偏移 10dp→0

  阶段4（2600-3000ms）:
    - 全屏渐隐（fade to 主界面）
    - 启动页 → 首页 转场: 交叉淡入 300ms

后续冷启动 — 精简版（1500ms）:

  阶段1（0-300ms）:
    - 背景: 直接显示完整渐变（无过渡）
    - 粒子: 简化为 20 个中心光点（非汇聚）
    - Logo: opacity=0, scale=0.8

  阶段2（300-900ms）:
    - Logo: opacity 0→1 (ease-out, 200ms), scale 0.8→1.0 (spring, 400ms)
    - 品牌名: opacity 0→1, Y偏移 10dp→0, duration=200ms, delay=100ms
    - 副标题: 同步淡入，delay=200ms

  阶段3（900-1200ms）:
    - Logo + 品牌名稳定展示 300ms

  阶段4（1200-1500ms）:
    - 全屏渐隐，交叉淡入首页，300ms
    - 点击可立即触发此阶段
```

## 三、全局动效系统

### 3.1 缓动曲线定义

| 名称 | 值 | 用途 |
|------|-----|------|
| **ease_standard** | `cubic-bezier(0.2, 0, 0, 1)` | 通用UI动画（开关、列表项） |
| **ease_enter** | `cubic-bezier(0, 0, 0.2, 1)` | 元素进入屏幕 |
| **ease_exit** | `cubic-bezier(0.4, 0, 1, 1)` | 元素退出屏幕 |
| **spring_snappy** | `stiffness: 400, damping: 28` | 快速弹性（按钮点击、Tab切换） |
| **spring_smooth** | `stiffness: 250, damping: 22` | 平滑弹性（卡片展开、页面转场） |
| **spring_bouncy** | `stiffness: 200, damping: 15` | 弹跳弹性（特效、庆祝动画） |
| **spring_subtle** | `stiffness: 300, damping: 30` | 微弱弹性（微交互、按压反馈） |

### 3.2 动画时长规范

| 类别 | 时长 | 说明 |
|------|------|------|
| **微交互** | 100-200ms | 按钮按压、开关切换、颜色过渡 |
| **标准过渡** | 200-300ms | 列表项出现、卡片状态切换、页面淡入淡出 |
| **页面转场** | 250-350ms | 页面间推入/推出、Tab切换 |
| **弹性动画** | 300-500ms | 带spring的展开/收起、特效 |
| **氛围动画** | 800-1500ms | 飘字、粒子效果、境界突破 |
| **全屏特效** | 2000-3000ms | 金字招牌冲屏、启动动画 |

### 3.3 震动反馈（Haptic）定义

| 场景 | 类型 | 时长 | 说明 |
|------|------|------|------|
| 按钮点击 | Light Impact | 10ms | 轻微确认 |
| 拍照快门 | Medium Impact | 20ms | 明确反馈 |
| 卡片消除 | Soft | 15ms | 轻柔反馈 |
| 对齐接近 | Selection | 5ms | 极短提示 |
| 相克惩罚 | Error | 200ms | 三段式：强-弱-弱 |
| 境界突破 | Success | 300ms | 渐强震动 |
| 金字招牌 | Heavy+Success | 400ms | 强击后渐弱 |
| 长按触发 | Warning | 50ms | 中等强度确认 |

### 5.2 无障碍

```yaml
最低要求:
  - 所有图标元素附带 contentDescription
  - 颜色不是唯一信息载体（文字+图标双重提示）
  - 支持系统字体大小调整（sp单位）
  - 支持 prefers-reduced-motion（减弱动效模式）

减弱动效模式:
  - 所有动画时长缩减至 50ms 或完全关闭
  - 粒子特效关闭
  - 震动反馈关闭
  - 仅保留必要的状态切换（开关、选中态）
```
